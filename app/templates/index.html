API Documentation

This API provides object detection using YOLO. Send a POST request to the /api/detect endpoint with an image file to get detection results.

Example Request

curl -X POST -F "file=@path/to/your/image.jpg" https://your-api-url/api/detect

Example Response

{
  "success": true,
  "detections": [
    {
      "class": "person",
      "confidence": 0.92,
      "bbox": [50, 120, 200, 380]
    },
    {
      "class": "car",
      "confidence": 0.87,
      "bbox": [300, 200, 500, 350]
    }
  ]
}
